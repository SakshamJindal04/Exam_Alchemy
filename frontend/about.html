Part A: The Project

Exam Alchemy

Transforming exam preparation by turning static PDFs into dynamic, AI-powered assessments.

Problem & Solution

Teachers spend hours manually creating question papers, while students lack personalized mock tests to practice effectively. Exam Alchemy solves this by using AI to parse existing course materials (like PDFs), extract questions, and dynamically generate balanced exam papers based on customizable difficulty parameters.

Target Users

Designed for Educators seeking an automated workflow for managing question banks and generating papers, and Students looking for a streamlined interface to practice sample papers and tests.

Core Features

  • AI Question Extraction: Extracts raw text from PDFs and passes it through the Google Gemini API to structure intelligent questions. This matters because it eliminates manual data entry for teachers.
  • Algorithmic Paper Generation: Balances Easy, Medium, and Hard question ratios to instantly generate customizable exam papers from the saved question bank.
  • Role-Based Access: Separate, tailored dashboard experiences for Teachers (creation & analytics) and Students (practice & taking tests).
  • Local Storage Sync: Saves generated banks and state locally, enabling a fast, stateless client-side experience for immediate feedback.

Tech Stack

Frontend Vanilla HTML5, CSS3, JavaScript
Backend Java Spring Boot, REST API
Database & Auth MongoDB, JWT Authentication
AI & Tools Google Gemini API, pdf.js
