## 🚀 Overview

Exam Alchemy is designed to streamline the examination process. It features a robust backend architecture, a responsive frontend, and containerized deployment for seamless scalability.

### ✨ Features
- **Frontend & Backend Separation:** Clean architecture for better maintainability.
- **Database Integration:** Reliable data storage for users, exams, and results.
- **Docker Support:** Easy local setup and deployment using Docker Compose.
- **Render Ready:** Configured for seamless deployment on Render.

## 📂 Project Structure

```
├── frontend/          # Client-side application
├── backend/           # Server and API endpoints
├── database/          # Database schemas and seed data
└── docker-compose.yml # Container orchestration
```

## 🛠️ Getting Started

### Prerequisites
Make sure you have [Docker](https://www.docker.com/) and [Docker Compose](https://docs.docker.com/compose/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SakshamJindal04/Exam_Alchemy.git
   cd Exam_Alchemy
   ```

2. **Run with Docker:**
   ```bash
   docker-compose up --build
   ```

3. **Access the application:**
   The frontend and backend will be available on their respective mapped ports (e.g., `localhost:3000` and `localhost:5000`).
